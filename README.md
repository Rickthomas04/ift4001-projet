# ift4001-projet
Projet de session du cours d'optimisation combinatoire
L'objectif de ce projet était d'explorer les capacités et les limites d'un solveur, pour apprendre quelque chose qui n'était pas présenté durant le cours sur les solveurs. Dans notre cas, nous avons choisi d'utiliser les nombres de Ramsey comme sujet pour accomplir ces tâches. Ce sujet devient rapidement complexe dû à la nature exponentielle de la taille des instances, ce qui force à trouver de nouvelles techniques dans le but de réduire le nombre de graphes total à traiter. Dans notre cas, nous avons tester un modèle d'optimisation locale pour essayer d'accélérer le traitement pour les grandes instances. De plus, nous avons aussi essayé des modèles plus traditionnels, chacun ayant leur forces et faiblesses. Tous nos résulats, notions apprises et utilisées ainsi que le protocole d'expérimentation suivi est décrit en détail dans notre rapport du projet.
